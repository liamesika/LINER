הפיכות — סיכום מפורט

הרצאות 17-19 · HW9 + HW10 · מותאם למבחני 2026
◆ ההגדרה הבסיסית + 4 התנאים השקולים הראשונים
הגדרה מטריצה הפיכה · A ∈ M_n(F) הפיכה אם קיימת B יחידה כך ש-AB = BA = I_n. נכתוב B = A⁻¹. הרצ' 17
משפט ⭐ A הפיכה ⟺ אחד מאלה מתקיים · 4 תנאים שקולים מרכזיים:
  1. det(A) ≠ 0  — "הנפח" של עמודות A אינו 0; A לא מקריסת מימדים.
  2. rank(A) = n  — A מנצלת את כל המימדים האפשריים.
  3. dim Nul(A) = 0,  כלומר  A·x̄ = 0̄  ⟹  x̄ = 0̄  — A לא "מאפסת" וקטור לא-טריוויאלי.
  4. עמודות (או שורות) A מהוות בסיס ל-F^n  — כל וקטור ב-F^n ניתן לייצוג יחיד.

פתק "det" — פונקצית הדטרמיננטה · כל הופעה של det בדף זה היא פונקצית הדטרמיננטה היחידה מ-M_n(F) → F שמקיימת: מולטי-לינארית בשורות + אלטרנטיבית + det(I_n) = 1. קיומה ויחידותה — משפט חשוב 3 (הרצ' 22). כשתשתמשי בה במבחן בהוכחה — חייב לציין "לפי משפט יחידות הדטרמיננטה".

★ ההוכחות הקריטיות — חובה לדעת למבחן
1. A הפיכה ⟺ עמודות בסיס ל-F^n
הוכחה דרך LC ו-Nul. הרצ' 17.
2. A הפיכה ⟺ Ax̄=b̄ פתיר יחיד
דו-כיווני: NulA = {0̄} ⟺ rank=n. הרצ' 17.
3. (AB)⁻¹ = B⁻¹A⁻¹
בדיקה ישירה: (AB)(B⁻¹A⁻¹) = I. הרצ' 17.
4. A הפיכה ⟺ det A ≠ 0
מטריצות אלמנטריות + מולטיפליקטיביות. הרצ' 23.
5. det(A⁻¹) = 1/det(A)
ממולטיפליקטיביות + det(I)=1. הרצ' 23.
6. AB הפיכה ⟹ A, B הפיכות
דרך rank: rank(AB) ≤ rank A, rank B. הרצ' 19.
7. AB = I_n (ריבועי) ⟹ BA = I_n
A "על" + ריבועי ⟹ הפיכה. HW9 Q6.
8. ∀B: BA=0 ⟹ B=0 ⟹ A הפיכה
בשלילה + בנייה של B מ-LC אפסי. HW10 Q5(b).
אסטרטגיה איך לבחור גישה במבחן?
  • להוכיח A הפיכה: בחרי תנאי שקול שהכי קל להוכיח (לרוב: det A ≠ 0, או Nul A = {0̄}, או rank A = n). ציטטי "לפי טבלת השקילויות".
  • לחשב A⁻¹: דירוג [A | I] או נוסחת 2×2 או נוסחה דרך det (לא בקורס).
  • להראות A לא הפיכה: det A = 0, או שורת/עמודת אפסים בקנונית, או עמודות ת"ל.
📚 סיכום הרצאה 21 — דטרמיננטה ופעולות שורה (לפי הסדר)
קונטקסט איפה אנחנו בקורס? · בהרצאה 20 הוגדרה det אקסיומטית (מולטי-לינארית + אלטרנטיבית + det(I_n)=1). הרצאה 21 בודקת איך det מתנהג תחת פעולות שורה, ומקבלת כתוצאה דרך לחשב det של מטריצה כללית. הרצאה 22 תוכיח יחידות + קיום ו-23 מולטיפליקטיביות.
תזכורת 1. שלוש האקסיומות של det
  1. מולטי-לינארית בכל שורה: אם משנים שורה i ל-α·u + β·v ושאר השורות זהות, אז det = α·det(...u...) + β·det(...v...).
  2. אלטרנטיבית: שתי שורות זהות ⟹ det = 0.
  3. נורמליזציה: det(I_n) = 1.
הרצ' 20
משפט (*) מרכזי 2. ⭐ פעולות שורה אלמנטריות ו-det · אם B מתקבלת מ-A ע"י פעולת שורה:
   • כפל שורה ב-c:  R_i → c·R_i  ⟹  det(B) = c · det(A)
   • החלפת שורות:  R_i ↔ R_j  ⟹  det(B) = −det(A)
   • הוספת כפולה:  R_i → R_i + α·R_j  ⟹  det(B) = det(A)
מבוסס על: מולטי-לינאריות (לכפל ולהוספת כפולה) + אלטרנטיביות (להחלפה ולהוספת כפולה).
מה אנחנו יודעים מעכשיו: ניתן לעקוב אחר det דרך כל דירוג שורה — det "צובר" קבועי תיקון.
למה 3. שורת אפסים ⟹ det = 0 · אם ל-A יש שורה R_i = 0̄^t, אז det(A) = 0.
מבוסס על: מולטי-לינאריות. R_i = 0̄ = 0·R_i ⟹ det = 0·det(A) = 0.
מה אנחנו יודעים: אם בדירוג מקבלים שורת אפסים — סיימנו, det = 0 מיידית.
למה 4. det(c·A_i) דרך שורה אחת · אם רק שורה i של A הוכפלה ב-c, אז det(B) = c · det(A).
מבוסס על: מולטי-לינאריות (מקרה פרטי של אקסיומה 1).
מה אנחנו יודעים: זה מקרה פרטי של משפט (*) — שורש המקרה הראשון.
למה 5. החלפת שורות הופכת סימן · R_i ↔ R_j ⟹ det(B) = −det(A).
מבוסס על: אלטרנטיביות + מולטי-לינאריות. הטריק: שיקלי det של מטריצה עם שורה R_i + R_j בשני המקומות — מתאפס, ופותחים לפי לינאריות.
מה אנחנו יודעים: כל החלפת שורות "עולה" ×(−1). עוקבים אחר מספר החלפות בחישוב.
למה 6. הוספת כפולה לא משנה det · R_i → R_i + α·R_j (i ≠ j) ⟹ det(B) = det(A).
מבוסס על: מולטי-לינאריות + אלטרנטיביות. פותחים לפי שורה i: det = det(...R_i...) + α·det(...R_j..., R_j...) = det(A) + α·0 = det(A).
מה אנחנו יודעים: הפעולה השכיחה ביותר בדירוג לא משנה det בכלל. כלומר אפשר לדרג לחופשי בלי לעקוב.
מסקנה 7. מטריצות שקולות שורה ו-det = 0 · אם A, B שקולות שורה, אז det(A) = 0 ⟺ det(B) = 0.
מבוסס על: משפט (*) — כל פעולת שורה משנה det בקבוע שונה מ-0. אם det היה 0, נשאר 0; אם לא היה — לא נופל ל-0.
מה אנחנו יודעים: "det = 0" היא תכונה שמורה תחת שקילות שורה. שימושי מאוד!
משפט 8. det של מטריצה משולשית · אם A משולשית (עליונה או תחתונה), אז det(A) = ∏_{i=1}^n [A]_{i,i}  (מכפלת איברי האלכסון).
מבוסס על: משפט (*) + אקסיומה det(I_n)=1. דרגי משולשית ל-I_n ע"י נורמליזציה (כפל כל שורה בהופכי) + ניקוי.
מה אנחנו יודעים: מטריצה משולשית — det = מכפלת האלכסון. אם איבר אלכסון אחד = 0 → det = 0.
משפט מרכזי חובה הוכחה 9. ⭐ A לא הפיכה ⟺ det(A) = 0
שקול: A הפיכה ⟺ det(A) ≠ 0.
מבוסס על: מסקנה 7 + משפט 8 + הגדרת הצורה הקנונית. הוכחה (בקצרה): דרגי A לצורה הקנונית C. אם A הפיכה: C = I_n ⟹ det(C) = 1 ≠ 0 ⟹ det(A) ≠ 0 (מסקנה 7). אם A לא הפיכה: ל-C יש שורת אפסים ⟹ det(C) = 0 ⟹ det(A) = 0.
מה אנחנו יודעים: זה אחד מ-13 התנאים השקולים להפיכות. הכי שימושי במבחנים — בדיקת הפיכות דרך det.
טכניקה 10. חישוב det דרך דירוג למשולשית
  1. דרגי את A למשולשית עליונה.
  2. עקבי: כל החלפת שורות → ×(−1). כל כפל שורה ב-c → det → c·det. הוספת כפולה → לא משנה.
  3. חשבי det של המשולשית = מכפלת האלכסון (משפט 8).
  4. "חזרי" את התיקונים: det(A) = (−1)^k · ∏(c_i)^{−1} · det(משולשית), כאשר k = מספר החלפות.
שימושי כש: n גדול (4×4 ומעלה) — מהיר יותר מפיתוח לפי שורה/עמודה.
