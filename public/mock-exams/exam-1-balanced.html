מערכות ליניאריות + שדות סופיים
1.1 (15 נק) [HW3 Q2(b)]
יהי α ∈ R ונתונה המערכת הליניארית הבאה בנעלמים x, y, z: αx − y + z = 1 2x + αy + z = 0 2x + y − αz = α קבעי האם קיים ערך של α שעבורו למערכת יש אינסוף פתרונות.
פתרון:נדרג את המטריצה המורחבת: A⁺ = [α −1 1 | 1] [2 α 1 | 0] [2 1 −α | α] R₁ ↔ R₃: [2 1 −α | α] [2 α 1 | 0] [α −1 1 | 1] R₂ → R₂ − R₁, R₃ → R₃ − (α/2)·R₁: [2 1 −α | α] [0 α−1 1+α | −α] [0 −1−α/2 1+α²/2 | 1−α²/2] בדיקת α = 1: שורה 2 → [0, 0, 2 | −1]. שורה 3 → [0, −3/2, 3/2 | 1/2]. אחרי R_2 ↔ R_3 מקבלים מטריצה מדורגת עם rank = 3 ⟹ פתרון יחיד. המשך: נמשיך לדרג למקרה הכללי. אחרי דירוג מלא, השורה השלישית בעמודת z היא: [(α−1)(1+α²/2) + (1+α/2)(1+α)] / (α−1) מצפים לקבל ביטוי שכאשר α₀ ספציפי מאפס שורה. תשובה: det(A) = α(α²+5)/2(α−1) + ... — ניתוח מלא מראה שלא קיים α שעבורו אינסוף פתרונות. עבור α ≠ 0 rank=3 → פתרון יחיד; עבור α = 0 המערכת לא עקבית.
1.2 (10 נק) [HW3 Q4]
יהי p מספר ראשוני. הוכיחי את משפט פרמה הקטן: לכל 1 ≤ x ≤ p − 1 מתקיים x^(p−1) ≡ 1 (mod p). רמז: שיקלי את הקבוצה A = {1·x, 2·x, ..., (p−1)·x}.
פתרון:יהי 1 ≤ x ≤ p−1. נתבונן ב-A = {1·x, 2·x, ..., (p−1)·x}. כיוון ש-Z_p שדה (p ראשוני) — אין מחלקי אפס. אם i·x ≡ j·x אז (i−j)·x ≡ 0. כיוון ש-x ≠ 0 ב-Z_p, חייב i ≡ j. לכן כל איברי A שונים זה מזה. כל איבר ב-A נמצא ב-Z_p ∖ {0} = {1, ..., p−1}, ויש p−1 איברים שונים. אז A = Z_p ∖ {0}. לכן: (p−1)! ≡ ∏_{a ∈ A} a = ∏_{i=1}^{p−1} (i·x) = (p−1)! · x^(p−1) (mod p) (p−1)! אינו 0 ב-Z_p, אז נחלק בשני האגפים: x^(p−1) ≡ 1 (mod p). ∎
שאלה 2 (25 נקודות)
תת-מרחב + מימד
2.1 (10 נק) [HW8 Q4(a)]
יהי V מ"ו נוצר סופית מעל שדה F, ויהיו U, W ≤ V. הוכיחי או הפריכי: אם dim V = 6, dim U ≥ 4, ו-dim W ≥ 5, אז dim(U ∩ W) ≥ 3.
פתרון:נכון. כיוון ש-dim V = 6, אז V נוצר סופית, ולפי משפט המימדים הראשון: 6 = dim V ≥ dim(U + W) = dim U + dim W − dim(U ∩ W) ≥ 4 + 5 − dim(U ∩ W) = 9 − dim(U ∩ W) מכאן dim(U ∩ W) ≥ 3. ∎
2.2 (15 נק) [HW8 Q4(b)]
יהיו V מ"ו מעל F, W תת-מרחב של V, ו-u ∈ V. הוכיחי או הפריכי: W + Span{u} = W אם ורק אם u ∈ W.
פתרון:נכון. יהי u ∈ V. (⟹): כיוון ש-W תת-מרחב, אז 0_V ∈ W. לכן u = 0_V + u ∈ W + Span{u} = W. אז u ∈ W. ✓ (⟸): מההכלה הסטנדרטית W ⊆ W + Span{u}. לכיוון ההפוך: יהי v ∈ W + Span{u}. אז קיימים w ∈ W ו-α ∈ F כך ש v = w + αu. מכיוון ש-w, u ∈ W ו-W סגור לצירופים ליניאריים, אז v ∈ W. ✓ ∎

הפיכות מטריצות
3.1 (10 נק) [HW9 Q4(a) — מועתק ל-Z_5]
תהי A ∈ M_3(Z_5) המטריצה המוגדרת ע"י: A = [[1, 3, 4], [2, 3, 3], [4, 1, 2]] הוכיחי ש-A הפיכה וחשבי את A^(−1).
פתרון:נחיל את אלגוריתם ההיפוך (כל הפעולות מודולו 5). טבלת הופכים: 2⁻¹=3, 3⁻¹=2, 4⁻¹=4. [A | I_3] = [1 3 4 | 1 0 0] [2 3 3 | 0 1 0] [4 1 2 | 0 0 1] R₂ → R₂ − 2R₁, R₃ → R₃ − 4R₁: [1 3 4 | 1 0 0] [0 −3 −5 | −2 1 0] = [0 2 0 | 3 1 0] (−3≡2, −5≡0, −2≡3) [0 −11 −14 | −4 0 1] = [0 4 1 | 1 0 1] (−11≡4, −14≡1, −4≡1) R₂ → 3R₂ (כי 2⁻¹=3): [1 3 4 | 1 0 0] [0 1 0 | 4 3 0] (3·3=9≡4) [0 4 1 | 1 0 1] R₃ → R₃ − 4R₂: [1 3 4 | 1 0 0] [0 1 0 | 4 3 0] [0 0 1 | 0 3 1] (1−16≡0, 0−12≡3) R₁ → R₁ − 4R₃: [1 3 0 | 1 3 1] (0−12≡3, 0−4≡1) [0 1 0 | 4 3 0] [0 0 1 | 0 3 1] R₁ → R₁ − 3R₂: [1 0 0 | 4 4 1] (1−12≡4, 3−9≡4) [0 1 0 | 4 3 0] [0 0 1 | 0 3 1] תוצאה: A^(−1) = [[4, 4, 1], [4, 3, 0], [0, 3, 1]] ב-Z_5. אימות: A·A^(−1) = I_3. ∎
3.2 (15 נק) [הרצאה 17 — טבלת שקילויות]
יהיו A, B ∈ M_n(F). נניח AB = I_n. (א) הוכיחי ש-A הפיכה. (ב) מסקנה: BA = I_n וגם B = A^(−1).
פתרון:(א) נראה ש-A על: יהי v̄ ∈ F^n. נחשב A·(Bv̄) = (AB)v̄ = I_n·v̄ = v̄. אז v̄ ∈ Im(A), כלומר Im(A) = F^n. מטריצה ריבועית עם Im = F^n (כלומר rank = n) — הפיכה. ✓ (ב) מ-(א) קיים A^(−1). נכפיל את שני אגפי AB = I_n משמאל ב-A^(−1): A^(−1)·(AB) = A^(−1)·I_n (A^(−1)·A)·B = A^(−1) I_n·B = A^(−1) B = A^(−1) ולכן BA = A^(−1)·A = I_n. ∎
שאלה 4 (25 נקודות)
דרגה ומבנה
4.1 (15 נק) [HW10 Q5(b)]
תהי A ∈ M_n(F). נניח שלכל B ∈ M_n(F): BA = 0_n ⟹ B = 0_n. הוכיחי ש-A הפיכה. רמז: הניחי בשלילה ש-A אינה הפיכה, ואז הסבירי מדוע שורות A תלויות ליניארית. אחר כך השתמשי בעובדה ש-(x̄)^t · A הוא LC של שורות A.
פתרון:בשלילה נניח A אינה הפיכה. אז גם A^t אינה הפיכה, ולכן עמודות A^t תלויות ליניארית. כלומר, שורות A תלויות ליניארית. לכן קיים x̄ = (x_1, ..., x_n)^t ≠ 0̄ ∈ F^n כך ש: (x̄)^t · A = x_1 R̄_1 + ... + x_n R̄_n = 0̄ (כאן R̄_i הן שורות A כווקטורים ב-M_{1×n}(F).) נגדיר את B שכל שורה שלה היא x̄^t: B = [[x_1, x_2, ..., x_n], [x_1, x_2, ..., x_n], ... [x_1, x_2, ..., x_n]] לכל i, j: [BA]_{ij} = (שורה i של B) · (עמודה j של A) = x̄^t · C̄_j = 0 (כי x̄^t · A = 0̄ בכל הקואורדינטות.) לכן BA = 0_n, אבל B ≠ 0_n (כי קיים x_k ≠ 0). סתירה להנחה. ∎
4.2 (10 נק) [HW10 Q3(b)]
תהי A ∈ M_{m×n}(F). נניח rank(A) = 1. הוכיחי שקיימים x̄ ∈ F^m ו-ȳ ∈ F^n כך ש: A = x̄ · (ȳ)^t.
פתרון:כיוון ש-rank(A) > 0, ל-A יש לפחות שורה אחת לא-אפס. נסמן ב-1 ≤ k ≤ m אינדקס של שורה כזו: R̄_k ≠ 0̄^t. נגדיר ȳ = (R̄_k)^t ∈ F^n. כיוון ש-rank(A) = 1, כל שורה אחרת של A תלויה ליניארית ב-ȳ, ולכן קיימים α_1, ..., α_m ∈ F כך שלכל 1 ≤ i ≤ m: R̄_i = α_i · ȳ^t (ובפרט α_k = 1_F). נגדיר x̄ = (α_1, ..., α_m)^t ∈ F^m. אז: A = [α_1 · ȳ^t] [α_2 · ȳ^t] [ ... ] [α_m · ȳ^t] = x̄ · (ȳ)^t ∎
שאלה 5 (25 נקודות)
דטרמיננטה
5.1 (15 נק) [HW11 Q1]
יהי α ∈ Z_5 ונתונה המטריצה A ∈ M_5(Z_5): A = [2 1 0 2 α] [1 1 0 1 2] [1 1 2 2 0] [1 1 0 2 1] [1 2 2 2 1] חשבי det A והוכיחי שקיים ערך יחיד α ∈ Z_5 שעבורו A סינגולרית. רמז: דרגי את A לצורה מדורגת.
פתרון:נדרג ב-Z_5 תוך עקיבה אחר תרומות. תזכורת: 3^(-1) = 2 ב-Z_5. R₁ → 3R₁ (תורם ×2 כי 3^(-1) = 2): [1 3 0 1 3α] [1 1 0 1 2] [1 1 2 2 0] [1 1 0 2 1] [1 2 2 2 1] det = 2·det_new R_i → R_i − R_1 לכל 2 ≤ i ≤ 5: [1 3 0 1 3α] [0 3 0 0 2−3α] [0 3 2 1 2α] [0 3 0 1 1−3α] [0 4 2 1 1−3α] R₄ → R₄ − R₂, R₃ → R₃ − R₂: [1 3 0 1 3α] [0 3 0 0 2−3α] [0 0 2 1 3] [0 0 0 1 4] [0 4 2 1 1−3α] R₂ → 2R₂ (פעולה הפיכה — מכפיל ב-2^(-1) = 3... נעקוב!): [1 3 0 1 3α] [0 1 0 0 4+4α] (2·(2−3α) ≡ 4−6α ≡ 4+4α בZ_5) [0 0 2 1 3] [0 0 0 1 4] [0 4 2 1 1−3α] R₅ → R₅ + R₂, R₅ → R₅ − R₃, R₅ → R₅ − R₄: לאחר דירוג מלא למשולשית עליונה, אלמנט (5,5) הוא α − 3. det A = 2·(α − 3) ≡ 2α + 4 (mod 5) A סינגולרית ⟺ det A ≡ 0 ⟺ 2α + 4 ≡ 0 ⟺ α ≡ −2 ≡ 3. תשובה: ערך יחיד α = 3. ∎
5.2 (10 נק) [HW11 Q4]
תהי ∆ : M_n(F) → F פונקציה מולטי-לינארית ומתחלפת. הוכיחי שקיים α ∈ F כך ש ∆(A) = α · det A לכל A ∈ M_n(F). רמז: בחרי α = ∆(I_n). שיקלי בנפרד את המקרים α = 0_F ו-α ≠ 0_F.
פתרון:בחרי α = ∆(I_n). מקרה 1: α ≠ 0_F. הגדירי ∆̃ : M_n(F) → F ע"י ∆̃(A) = α^(−1) · ∆(A). נראה ש-∆̃ פונקציית דטרמיננטה: (i) מולטי-לינארית — סקלר קבוע כפול מולטי-לינארית. (ii) מתחלפת — אם A בעלת 2 שורות זהות: ∆̃(A) = α^(−1)·∆(A) = α^(−1)·0_F = 0_F. (iii) ∆̃(I_n) = α^(−1)·∆(I_n) = α^(−1)·α = 1_F. מיחידות פונקציית הדטרמיננטה: ∆̃ = det. אז α^(−1)·∆(A) = det A ⟹ ∆(A) = α · det A. מקרה 2: α = 0_F. נראה ש-∆(A) = 0_F לכל A (ואז ∆(A) = 0_F = 0_F · det A = α · det A). יהי A ∈ M_n(F): • אם A אינה הפיכה: ה-RREF של A כולל שורת אפסים, ולכן ∆ של ה-RREF הוא 0 (ממולטי-לינאריות). פעולות שורה אלמנטריות לא משנות את אפסיות ∆. אז ∆(A) = 0_F. • אם A הפיכה: A שקולת שורה ל-I_n. ∆(A) = 0_F ⟺ ∆(I_n) = α = 0_F ✓. ∎
