שאלה 1 (25 נקודות)
משפט הדרגה (Rank-Nullity)
1.1 (5 נק)
תהי A ∈ M_{m×n}(F). הגדירי: (א) rank A (ב) NulA
פתרון:(א) rank A מוגדר כמספר המקדמים המובילים בצורה המדורגת הקנונית של A. (ב) NulA = {x̄ ∈ F^n : A·x̄ = 0̄} — מרחב האפס של A, תת-מרחב של F^n.
1.2 (15 נק)
תהי A ∈ M_{m×n}(F). הוכיחי משפט הדרגה (Rank-Nullity): rank A + dim NulA = n.
פתרון:נסמן k = rank A. נדרג את A לצורתה הקנונית C. יהיו עמודות מובילות באינדקסים j_1 < ... < j_k, ועמודות לא-מובילות (משתנים חופשיים) באינדקסים i_1 < ... < i_{n-k}. בניית בסיס ל-NulA = NulC (פעולות שורה לא משנות NulA): לכל ℓ ∈ {1, ..., n−k} נגדיר v̄_ℓ ∈ F^n כך ש: • (v̄_ℓ)_{i_ℓ} = 1 • (v̄_ℓ)_{i_p} = 0 לכל p ≠ ℓ • המשתנים הקשורים נקבעים מהמשוואות. בת"ל: נניח Σ α_ℓ v̄_ℓ = 0̄. בקואורדינטה i_p: רק v̄_p תורמת 1, האחרים 0. אז α_p = 0 לכל p. פורש: יהי x̄ ∈ NulA. נציב את ערכי המשתנים החופשיים של x̄ במקדמי v̄_ℓ — מקבלים x̄ = Σ x_{i_ℓ} · v̄_ℓ. מסקנה: dim NulA = n − k, ולכן rank A + dim NulA = k + (n−k) = n. ∎
1.3 (5 נק)
תהי A ∈ M_{m×n}(F) עם m < n. הוכיחי שלמערכת A·x̄ = 0̄ קיים פתרון לא טריוויאלי.
פתרון:rank A ≤ m < n. ממשפט הדרגה: dim NulA = n − rank A ≥ n − m > 0. אז NulA ⊋ {0̄}, כלומר קיים x̄ ≠ 0̄ ב-NulA. ∎
שאלה 2 (25 נקודות)
טבלת שקילויות ההפיכות
2.1 (10 נק)
יהי F שדה ו-A ∈ M_n(F). הוכיחי: A הפיכה אם ורק אם לכל b̄ ∈ F^n למערכת A·x̄ = b̄ קיים פתרון יחיד.
פתרון:(⟹): נניח A הפיכה. יהי b̄ ∈ F^n. נציב x̄ = A^(−1)·b̄ — מקיים A·x̄ = b̄. יחידות: אם x̄_1, x̄_2 פתרונות, אז A(x̄_1 − x̄_2) = 0̄ ⟹ x̄_1 − x̄_2 ∈ NulA = {0̄} (כי A הפיכה) ⟹ x̄_1 = x̄_2. ✓ (⟸): בפרט, A·x̄ = 0̄ פתיר יחיד x̄ = 0̄. אז NulA = {0̄}. ממשפט הדרגה: rank A = n. A ריבועית עם rank=n ⟺ הפיכה. ∎
2.2 (15 נק)
יהי A ∈ M_n(F). הוכיחי: A הפיכה אם ורק אם עמודות A מהוות בסיס של F^n.
פתרון:נסמן את עמודות A ב-C̄_1, ..., C̄_n. תזכורת: A·x̄ = x_1 C̄_1 + ... + x_n C̄_n. (⟹): נניח A הפיכה. בת"ל: אם Σ α_i C̄_i = 0̄, אז A·ᾱ = 0̄ (כאשר ᾱ = (α_1,...,α_n)^t) ⟹ ᾱ = 0̄ ⟹ כל α_i = 0. פורשים: יהי v̄ ∈ F^n. נציב ᾱ = A^(−1)·v̄ ⟹ A·ᾱ = v̄, כלומר v̄ = Σ α_i C̄_i. ✓ (⟸): נניח {C̄_i}_{i=1}^n בסיס. אז A·x̄ = 0̄ ⟺ Σ x_i C̄_i = 0̄ ⟺ (מבת"ל) x̄ = 0̄. אז NulA = {0̄}, וממשפט הדרגה: rank A = n ⟹ A הפיכה. ∎
שאלה 3 (25 נקודות)
דטרמיננטה — חישוב + הוכחה
3.1 (8 נק) [HW12 Q1(a)]
חשבי את הדטרמיננטה של: A = [2 1 4 3] [−1 0 2 2] [4 0 1 9] [6 0 4 2] רמז: פתחי את הדטרמיננטה לפי העמודה השנייה.
פתרון:בעמודה השנייה רק [A]_{12} = 1 שונה מ-0 (האחרים אפסים). פיתוח: det A = (−1)^{1+2} · 1 · M_{12}^{(A)} = −M_{12}^{(A)} M_{12}^{(A)} = det[−1 2 2] [ 4 1 9] [ 6 4 2] לפי הנוסחה הישירה ל-3×3: = (−1)(1)(2) + (2)(9)(6) + (2)(4)(4) − (6)(1)(2) − (4)(9)(−1) − (2)(4)(2) = −2 + 108 + 32 − 12 + 36 − 16 = 146 לכן det A = −146. ∎
3.2 (17 נק)
יהיו A, B ∈ M_n(F). הוכיחי משפט המולטיפליקטיביות של הדטרמיננטה: det(AB) = det(A) · det(B).
פתרון:מקרה 1: B אינה הפיכה. rank B < n, אז rank(AB) ≤ rank B < n, ולכן AB גם אינה הפיכה. אז det(AB) = 0 = det A · 0 = det A · det B. ✓ מקרה 2: B הפיכה. B מתפרקת למכפלת מטריצות אלמנטריות: B = E_1 · E_2 · ... · E_k. למה (נדרשת לציטוט): לכל E אלמנטרית ולכל A ∈ M_n(F): det(EA) = det(E) · det(A) הוכחת הלמה: 3 סוגי פעולות: • R_i → cR_i: det(EA) = c · det A, det E = c. ✓ • R_i ↔ R_j: det(EA) = −det A, det E = −1. ✓ • R_i → R_i + αR_j: det(EA) = det A, det E = 1. ✓ נשתמש בלמה k פעמים: det(AB) = det(A · E_1 · ... · E_k) נכפיל מימין בכל אלמנטרית בנפרד (סדר חשוב!): det(A · E_1 · ... · E_k) = det(A · E_1 · ... · E_{k−1}) · det(E_k) = det(A · E_1 · ... · E_{k−2}) · det(E_{k−1}) · det(E_k) = ... = det A · det(E_1) · ... · det(E_k) גם det B = det(E_1 · ... · E_k) = det(E_1) · ... · det(E_k). לכן det(AB) = det A · det B. ∎
שאלה 4 (25 נקודות)
בסיס + מימד
4.1 (10 נק)
הוכיחי את משפט השלוש: יהי V מ"ו עם dim V = n ויהיו v_1, ..., v_n ∈ V. הוכיחי שהבאים שקולים: (א) {v_1, ..., v_n} בסיס של V. (ב) {v_1, ..., v_n} בת"ל. (ג) {v_1, ..., v_n} פורשים את V.
פתרון:(א) ⟹ (ב): מהגדרת בסיס. (ב) ⟹ (ג) בשלילה: נניח {v_i} בת"ל אך לא פורש. אז קיים v ∈ V ∖ Span{v_1,...,v_n}. נראה ש-{v_1, ..., v_n, v} בת"ל: יהיו α_1, ..., α_n, β ∈ F כך ש α_1 v_1 + ... + α_n v_n + β v = 0_V. • אם β ≠ 0: v = −β^(−1)(α_1 v_1 + ... + α_n v_n) ∈ Span{v_1,...,v_n} — סתירה. • אז β = 0. אז α_1 v_1 + ... + α_n v_n = 0_V, ומבת"ל הראשונית כל α_i = 0. מצאנו n+1 וקטורים בת"ל ב-V — סתירה למסקנה ש-dim V = n ⟹ כל n+1 וקטורים ת"ל. ∎ (ג) ⟹ (א) בשלילה: נניח פורשים אבל ת"ל. אז קיים j כך ש v_j ∈ Span{v_1,...,v_{j−1}, v_{j+1},...,v_n}. נסיר את v_j — הקבוצה עדיין פורשת. מצאנו n−1 וקטורים פורשים — סתירה למסקנה ש-dim V = n ⟹ n−1 וקטורים אינם פורשים. ∎
4.2 (15 נק)
הוכיחי את משפט המימדים הראשון: יהי V מ"ו נוצר סופית מעל F ויהיו U, W תתי-מרחב של V. הוכיחי: dim(U + W) = dim U + dim W − dim(U ∩ W).
פתרון:נסמן d = dim(U ∩ W), ויהי B_0 = {u_1, ..., u_d} בסיס ל-U ∩ W. ממשפט ההשלמה ניתן להשלים את B_0 לבסיס של U: B_U = {u_1, ..., u_d, x_1, ..., x_p}, dim U = d + p. ובאופן דומה לבסיס של W: B_W = {u_1, ..., u_d, y_1, ..., y_q}, dim W = d + q. טענה: B = {u_1,...,u_d, x_1,...,x_p, y_1,...,y_q} בסיס ל-U + W. פורש: לכל v ∈ U + W: v = u + w כאשר u LC של B_U ו-w LC של B_W. אז v LC של B. ✓ בת"ל: נניח Σ α_i u_i + Σ β_j x_j + Σ γ_k y_k = 0_V. נעביר אגף: Σ γ_k y_k = −(Σ α_i u_i + Σ β_j x_j). צד ימין ב-U. צד שמאל ב-W (LC של y_k). לכן Σ γ_k y_k ∈ U ∩ W, וניתן לכתוב כ-LC של {u_i} בלבד: Σ γ_k y_k = Σ δ_i u_i. אבל B_W = {u_i, y_k} בת"ל, אז γ_k = 0 לכל k. נשאר Σ α_i u_i + Σ β_j x_j = 0_V. B_U = {u_i, x_j} בת"ל, אז α_i = β_j = 0 לכל i, j. ✓ גודל: |B| = d + p + q = (d + p) + (d + q) − d = dim U + dim W − dim(U ∩ W). ∎
שאלה 5 (25 נקודות)
בלוקים + תלת-אלכסונית
5.1 (15 נק) [HW12 Q3]
יהי A ∈ M_n(R). נתבונן במטריצות הבלוקים U, K ∈ M_{2n}(R) המוגדרות: U = [I_n −A] [0_n I_n] , K = [I_n A] [A I_n] (א) הוכיחי כי U · K = [I_n − A² 0_n] [A I_n ] (ב) נניח det(K) = 180 ו-det(I_n − A) = −6. חשבי det(I_n + A). רמז: השתמשי בתכונה המולטיפליקטיבית של det.
פתרון:(א) כפל בלוקים: UK = [I_n·I_n + (−A)·A I_n·A + (−A)·I_n] [0_n·I_n + I_n·A 0_n·A + I_n·I_n] = [I_n − A² 0_n] [A I_n] ✓ (ב) U בלוקים משולשית עליונה עם I על האלכסון: det U = det(I_n) · det(I_n) = 1 UK בלוקים משולשית תחתונה: det(UK) = det(I_n − A²) · det(I_n) = det(I_n − A²) ממולטיפליקטיביות: det(UK) = det U · det K = 1 · 180 = 180 אז det(I_n − A²) = 180. אבל I_n − A² = (I_n − A)(I_n + A): det(I_n − A²) = det(I_n − A) · det(I_n + A) = (−6) · det(I_n + A) 180 = −6 · det(I_n + A) ⟹ det(I_n + A) = −30. ∎
5.2 (10 נק) [HW12 Q2(b)]
תהי A_n ∈ M_n(R) תלת-אלכסונית עם 2 על האלכסון ו-(−1) מעל ומתחת לאלכסון: A_n = [ 2 −1 0 ... 0 0] [−1 2 −1 ... 0 0] [ 0 −1 2 ... 0 0] [... ... ... ... ... ...] [ 0 0 0 ... 2 −1] [ 0 0 0 ... −1 2] הוכיחי באינדוקציה חזקה: det(A_n) = n + 1 לכל 2 ≤ n ∈ N.
פתרון:בסיסים: n = 2: det A_2 = det[2 −1; −1 2] = 4 − 1 = 3 = 2 + 1. ✓ n = 3: det A_3 = 2(4−1) − (−1)(−2−0) = 6 − 2 = 4 = 3 + 1. ✓ צעד אינדוקציה: יהי n ≥ 4 וניחה ש det(A_k) = k+1 לכל 2 ≤ k ≤ n−1. בפרט det A_{n−1} = n ו-det A_{n−2} = n−1. נחשב det A_n ע"י פיתוח לפי שורה ראשונה: det A_n = 2 · M_{11}^{(A_n)} − (−1) · M_{12}^{(A_n)} = 2·M_{11} + M_{12} חישוב M_{11}: מחיקת שורה 1 ועמודה 1 → בדיוק A_{n−1}. אז M_{11} = det A_{n−1} = n. חישוב M_{12}: מחיקת שורה 1 ועמודה 2. השורה הראשונה החדשה: (−1, −1, 0, ..., 0). נפתח לפי עמודה 1: M_{12} = (−1) · det A_{n−2} = −(n−1) סיכום: det A_n = 2n + (−(n−1)) = 2n − n + 1 = n + 1 ✓ ∎
