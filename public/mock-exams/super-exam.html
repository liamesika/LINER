שאלה 1 (10 נקודות) — משפט הדרגה
הרצאה 19 · סבירות 75%
1 (10 נק)
תהי A ∈ M_{m×n}(F). (א) (2 נק) הגדירי את rank A ואת Nul A. (ב) (8 נק) הוכיחי: rank A + dim Nul A = n.
פתרון:(א) rank A = מספר המקדמים המובילים בצורה המדורגת הקנונית של A. Nul A = {x̄ ∈ F^n : A·x̄ = 0̄} — תת-מרחב של F^n. (ב) נסמן k = rank A. נדרג את A ל-RREF, נסמנה C. יהיו עמודות מובילות באינדקסים j_1 < ... < j_k, ועמודות חופשיות באינדקסים i_1 < ... < i_{n-k}. בניית בסיס ל-Nul A: לכל ℓ ∈ {1,...,n-k}, נגדיר v̄_ℓ ∈ F^n כך ש: • (v̄_ℓ)_{i_ℓ} = 1 • (v̄_ℓ)_{i_p} = 0 לכל p ≠ ℓ • המשתנים הקשורים נקבעים מהמשוואות של C. בת"ל: אם Σ α_ℓ v̄_ℓ = 0̄, נסתכל על קואורדינטה i_p: רק v̄_p תורם 1 שם, האחרים 0. אז α_p = 0 לכל p. פורש את Nul A: יהי x̄ ∈ Nul A. נציב את ערכי המשתנים החופשיים של x̄ כמקדמים, ונקבל x̄ = Σ x_{i_ℓ} · v̄_ℓ. מסקנה: dim Nul A = n - k, ולכן rank A + dim Nul A = k + (n-k) = n. ∎
שאלה 2 (10 נקודות) — משפט השלוש
הרצאה 15 · סבירות 80%
2 (10 נק)
יהי V מ"ו עם dim V = n ויהיו v_1, ..., v_n ∈ V. הוכיחי שהבאים שקולים: (א) {v_1,...,v_n} בסיס של V. (ב) {v_1,...,v_n} בת"ל. (ג) {v_1,...,v_n} פורשים את V.
פתרון:(א) ⟹ (ב): מהגדרת בסיס. (ב) ⟹ (ג) בשלילה: נניח {v_i} בת"ל אך לא פורש. אז קיים v ∈ V ∖ Span{v_1,...,v_n}. נראה ש-{v_1,...,v_n,v} בת"ל: יהיו α_1,...,α_n,β ∈ F כך ש α_1 v_1 + ... + α_n v_n + β v = 0_V. • אם β ≠ 0: v = -β^(-1)·Σ α_i v_i ∈ Span{v_i} — סתירה. • אז β = 0, ומבת"ל הראשונית: כל α_i = 0. מצאנו n+1 וקטורים בת"ל — סתירה למסקנה ש-dim V = n ⟹ כל n+1 וקטורים ת"ל. (ג) ⟹ (א) בשלילה: נניח פורשים אך ת"ל. קיים j: v_j ∈ Span{v_1,...,v_{j-1},v_{j+1},...,v_n}. נסיר את v_j — הקבוצה עדיין פורשת. אז יש n-1 וקטורים פורשים — סתירה ל-dim V = n ⟹ n-1 וקטורים אינם פורשים. ∎
שאלה 3 (10 נקודות) — שקילויות הפיכות
הרצאה 17 · סבירות 75%
3 (10 נק)
יהי F שדה ו-A ∈ M_n(F). (א) (5 נק) הוכיחי: A הפיכה ⟺ עמודות A בסיס ל-F^n. (ב) (5 נק) הוכיחי: A הפיכה ⟺ לכל b̄ ∈ F^n, Ax̄ = b̄ יש פתרון יחיד.
פתרון:(א) נסמן C̄_1,...,C̄_n את עמודות A. תזכורת: A·x̄ = x_1 C̄_1 + ... + x_n C̄_n. (⟹) A הפיכה: • בת"ל: Σα_i C̄_i = 0̄ ⟺ A·ᾱ = 0̄ ⟹ ᾱ = 0̄. • פורש: v̄ ∈ F^n ⟹ ᾱ = A^(-1)·v̄ מקיים A·ᾱ = v̄. (⟸) עמודות בסיס ⟹ Ax̄ = 0̄ ⟺ Σx_i C̄_i = 0̄ ⟺ x̄ = 0̄ (בת"ל) ⟹ Nul A = {0̄} ⟹ ממשפט הדרגה rank A = n ⟹ A הפיכה. (ב) (⟹) A הפיכה: x̄ = A^(-1)b̄ פתרון. יחידות: שני פתרונות x̄_1, x̄_2 ⟹ A(x̄_1 - x̄_2) = 0̄ ⟹ x̄_1 = x̄_2 (כי Nul A = {0̄}). (⟸) בפרט Ax̄ = 0̄ יחיד: x̄ = 0̄ ⟹ Nul A = {0̄} ⟹ rank A = n ⟹ הפיכה. ∎
שאלה 4 (10 נקודות) — det(AB) = det(A)·det(B)
הרצאה 22 · סבירות 65%
4 (10 נק)
יהיו A, B ∈ M_n(F). הוכיחי משפט המולטיפליקטיביות: det(AB) = det(A) · det(B).
פתרון:מקרה 1: B אינה הפיכה. rank B < n ⟹ rank(AB) ≤ rank B < n ⟹ AB אינה הפיכה ⟹ det(AB) = 0. גם det B = 0. אז det(AB) = 0 = det A · 0 = det A · det B. ✓ מקרה 2: B הפיכה. B מתפרק למכפלת מטריצות אלמנטריות: B = E_1 · E_2 · ... · E_k. למה: לכל E אלמנטרית ולכל A ∈ M_n(F): det(EA) = det(E) · det(A) (הוכחה: 3 סוגי פעולות — כפל שורה ב-c: det(E)=c; החלפת שורות: det(E)=-1; הוספת כפולה: det(E)=1.) נשתמש בלמה k פעמים: det(AB) = det(A·E_1·...·E_k) = det(A·E_1·...·E_{k-1}) · det(E_k) = ... = det A · det(E_1) · ... · det(E_k) גם det B = det(E_1·...·E_k) = det(E_1)·...·det(E_k). לכן det(AB) = det A · det B. ∎
שאלה 5 (10 נקודות) — למת שטיינל
הרצאות 11-12 · סבירות 70%
5 (10 נק)
הוכיחי את למת ההחלפה של שטיינל: יהי V מ"ו, ויהיו v_1,...,v_k בסיס של V ו-w_1,...,w_m פורשים את V. אז k ≤ m. הסיקי: כל שני בסיסים של V הם באותו גודל.
פתרון:בשלילה: נניח k > m. שלב 1: {w_1,...,w_m} פורש, אז v_1 = Σ α_j w_j. v_1 ≠ 0_V (כי בבסיס), אז קיים α_{j_0} ≠ 0. נחליף w_{j_0} ב-v_1: w_{j_0} = α_{j_0}^(-1)·(v_1 - Σ_{j ≠ j_0} α_j w_j) אז S_1 = {v_1, w_1,...,ŵ_{j_0},...,w_m} עדיין פורש. שלב i (אינדוקציה): נניח S_{i-1} = {v_1,...,v_{i-1}, w_{σ(1)},...,w_{σ(m-i+1)}} פורש. נכתוב v_i כ-LC של S_{i-1}. המקדמים של v_1,...,v_{i-1} לא יכולים להיות כל המקדמים (אחרת v_i ∈ Span{v_1,...,v_{i-1}} — סתירה לבת"ל של בסיס). אז יש מקדם ≠ 0 של איזשהו w_σ. נחליף — מקבלים S_i פורש. שלב m: S_m = {v_1,...,v_m} פורש את V. כיוון k > m, יש v_{m+1} בבסיס. אז v_{m+1} ∈ Span{v_1,...,v_m} — סתירה לבת"ל של הבסיס. ∎ מסקנה: אם B_1 (גודל k) ו-B_2 (גודל m) שני בסיסים, אז: • B_1 בסיס, B_2 פורש ⟹ k ≤ m. • B_2 בסיס, B_1 פורש ⟹ m ≤ k. מסקנה: k = m. ∎
שאלה 6 (10 נקודות) — מערכת תלוית פרמטר

פתרון:נדרג את המורחבת: A⁺ = [1 2 a | 1] [2 a 2 | a] [a 2 1 | a] R₂ → R₂ - 2R₁, R₃ → R₃ - a·R₁: [1 2 a | 1 ] [0 a-4 2-2a | a-2 ] [0 2-2a 1-a² | a-a ] = [0 2-2a 1-a² | 0] R₃ → R₃ + ((2-2a)/(a-4))·R₂ (כש-a ≠ 4): מחשבים... אחרי דירוג מלא, שורה 3 → [0, 0, P(a) | Q(a)] כאשר P, Q פולינומים ב-a. ניתוח: • a = 4: שורה 2 → [0, 0, -6 | 2]. בודקים את שורה 3. • a = 1: 2-2a = 0. שורה 3 → [0, 0, 0 | 0]. אם שורה 2 גם מתאפסת — אינסוף; אחרת יחיד. • a = -1: 1-a² = 0. בודקים. תשובה כללית: פתרון יחיד ⟺ det A ≠ 0. נחשב det A: det A = 1·(a-4) - 2·(2-2a) + a·(4-a²) = a-4 - 4+4a + 4a-a³ = -a³ + 9a - 8 -a³+9a-8 = -(a³-9a+8) = -(a-1)(a²+a-8) שורשים: a = 1 ו-a = (-1±√33)/2. מסקנה: פתרון יחיד עבור a ∉ {1, (-1+√33)/2, (-1-√33)/2}. עבור הערכים האלו — בודקים בנפרד אם אין/אינסוף. ∎

פתרון:(א) מסדרים את 4 הוקטורים כשורות ומדרגים: [1 2 0] [0 1 1] [1 0 1] [0 3 2] R₃ → R₃ - R₁: [1 2 0] [0 1 1] [0 -2 1] [0 3 2] R₃ → R₃ + 2R₂, R₄ → R₄ - 3R₂: [1 2 0] [0 1 1] [0 0 3] [0 0 -1] R₄ → R₄ + R₃/3: [1 2 0] [0 1 1] [0 0 3] [0 0 0] בסיס ל-U+W: {(1,2,0), (0,1,1), (0,0,1)}. dim(U+W) = 3 (כלומר U+W = R^3). dim U = 2, dim W = 2 (כל זוג וקטורים לא כפולות). ממשפט המימדים: dim(U ∩ W) = 2 + 2 - 3 = 1 לבסיס של U ∩ W: ניקח s(1,2,0) + t(0,1,1) = α(1,0,1) + β(0,3,2). פותרים: s = α, 2s + t = 3β, t = α + 2β. אחרי חישוב: U ∩ W = Span{(1,3,2)} (לדוגמה). (ב) ממשפט המימדים: dim(U+W) = 5 + 4 - dim(U∩W) = 9 - dim(U∩W). U+W ⊆ V ⟹ dim(U+W) ≤ 7. לכן 9 - dim(U∩W) ≤ 7 ⟹ dim(U∩W) ≥ 2. ∎

שאלה 10 (10 נקודות) — rank(A) = 1
HW10 Q3(b) · סבירות 50%
10 (10 נק)
תהי A ∈ M_{m×n}(F) עם rank A = 1. הוכיחי שקיימים x̄ ∈ F^m ו-ȳ ∈ F^n כך ש: A = x̄ · (ȳ)^t.
פתרון:כיוון rank A = 1 > 0, יש ל-A לפחות שורה אחת לא-אפס. נסמן ב-1 ≤ k ≤ m אינדקס של שורה כזו: R̄_k ≠ 0̄^t. נגדיר: ȳ = (R̄_k)^t ∈ F^n כיוון ש-rank A = 1, dim Row A = 1, ולכן ȳ^t פורש את Row A. אז כל שורה R̄_i היא כפולה של ȳ^t: R̄_i = α_i · ȳ^t לאיזשהו α_i ∈ F (ובפרט α_k = 1_F). נגדיר: x̄ = (α_1, α_2, ..., α_m)^t ∈ F^m נחשב את המכפלה x̄ · ȳ^t: [x̄ · ȳ^t]_{ij} = x̄_i · ȳ_j = α_i · ȳ_j זה בדיוק האיבר ה-(i,j) של A, כי שורה i של A היא α_i · ȳ^t. מסקנה: A = x̄ · (ȳ)^t ∎
